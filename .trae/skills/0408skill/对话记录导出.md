## 理解参数概念
**助手：** 为什么用 `name` 而不是"小叽"？

- `name` 是一个**参数**，就像一个"盒子"，可以装不同的名字
- 如果你写"小叽"，那这个函数只能对小叽说"你好"
- 如果你用 `name`，这个函数可以对任何人说"你好"

**助手：** 现在你试试：

**题目：** 用 `greet` 函数对"小明"、"小红"、"张三"说"你好"

**用户：** 名字

## 详细解释参数
**助手：** 让我更清楚地解释一下：

## 当你调用 `greet("小明")` 时发生了什么：

1. **`"小明"`** 被传给函数
2. 在函数内部，**`name`** 变量的值变成了 **`"小明"`**
3. 所以 `print("你好，" + name + "！")` 会输出：**`你好，小明！`**

## 同样的：

- `greet("小明")` → 输出：`你好，小明！`
- `greet("小红")` → 输出：`你好，小红！`
- `greet("张三")` → 输出：`你好，张三！`

## 关键点：

**`name`** 只是一个"占位符"，真正使用函数时，你传入什么名字，它就变成什么名字。

**用户：** 有点不明白

## 信封比喻
**助手：** 没关系！让我用更简单的方式解释。

## 想象一个"信封"：

**函数定义时：**
```python
def greet(name):
    print("你好，" + name + "！")
```
- `name` 就像一个**空信封**，等待你往里面放东西

**调用函数时：**
```python
greet("小明")
```
- 你把 **"小明"** 放进了这个信封里
- 现在信封里装的是"小明"

**函数执行时：**
```python
print("你好，" + name + "！")
```
- 函数打开信封，看到里面是"小明"
- 所以输出：**`你好，小明！`**

## 再举一个例子：

```python
def say_hello(信封):
    print("你好，" + 信封 + "！")

say_hello("小明")  # 把"小明"放进信封
say_hello("小红")  # 把"小红"放进信封
```

**用户：** 嗯嗯明白了
